Firmware for 802.11 devices
===========================

.. toctree::

   firmware/openfirmware

Firmware for 802.11 devices are found on the `linux-firmware git tree
<https://git.kernel.org/cgit/linux/kernel/git/firmware/linux-firmware.git/>`__.
Some firmware binaries have source code available for it, for details on
that refer to the :doc:`open firmware <firmware/openfirmware>` page.
