Official Linux Wireless documentation
=====================================

This is the main source of Documentation for the Linux wireless
(IEEE-802.11) subsystem. This wiki features information for end-users,
developers and vendors.

This site covers all the new 802.11 :doc:`drivers <en/users/drivers>`,
the new core :doc:`mac80211 <en/developers/documentation/mac80211>` and
:doc:`cfg80211 <en/developers/documentation/cfg80211>` components along
with the new userspace and in-kernel :doc:`nl80211
<en/developers/documentation/nl80211>` configuration interface.
:doc:`New nl80211 userspace <en/users/documentation>` applications are
also documented.
